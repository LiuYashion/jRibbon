jRibbon
===================


一个简单的JQ插件，用以实现div的丝带包角效果。

    <script type="text/javascript" src="jRibbon.min.js" ></script>

    $('#test').jRibbon({
		'frontColor': 'red',	// 包角丝带的正面颜色
		'backColor': 'brown'	// 包角丝带的背面颜色
	})
